# batch-stub-mixpanel
Stub mixpanel object

## Installation
```sh
$ npm install batch-stub-mixpanel
```

## Usage
To stub mixpanel object after mixpanel library is introduced:
```js
var batchStubMixpanel = require('batch-stub-mixpanel');
batchStubMixpanel(mixpanel);
```
